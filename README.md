# Angular + Capacitor Playground

An exploration project for testing Angular with Capacitor for cross-platform mobile development.

## What's being explored

- Angular + Capacitor integration and build flow
- Native plugin access (camera, filesystem, etc.)
- Authentication patterns
- Local and native storage
- Push notifications
- Deploying to Android via Capacitor

## Getting started

```bash
npm install
npm run dev # web dev server
npm run build # production build
npx cap sync # sync web assets to native projects
npx cap open android # open in Android Studio
```
